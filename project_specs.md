# 프로젝트 명세 및 작업 추적

## 프로젝트 정보
- **프로젝트명**: AI 가계부 - Windows 애플리케이션
- **시작일**: 2026-01
- **목표**: AI 기능이 포함된 윈도우 데스크톱 가계부 애플리케이션 개발

## 기술 스택
- **프론트엔드**: Electron + React + TypeScript + Tailwind CSS
- **백엔드**: Spring Boot + SQLite + JPA
- **AI 서비스**: Python FastAPI + OpenAI API / Local LLM
